# SPEC

NFER Reading Builder is a static Year 4 reading assessment generator hosted on GitHub Pages.

## Functional requirements
- Generate tests with 2 passages (1 fiction, 1 nonfiction) and 12 questions.
- Target domain split: retrieval 3, vocabulary 2, inference 4, structure 1, author intent 2.
- Mark responses client-side.
- Produce diagnostic and recommendations.
- Track progress in LocalStorage with adaptive difficulty rules.
- Support print views for pupil, teacher and diagnostics.
