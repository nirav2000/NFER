# Year 4 Reading Unit Quality Checklist

This checklist must be completed before a unit is added to the master JSON library.

---

# Passage Quality

- Passage length between **300–400 words**
- Written in **clear British English**
- Vocabulary appropriate for **Year 4 readers**
- No repeated filler phrases such as:
  - "one morning"
  - "for breakfast"
  - "in Year 4"
- Topic clearly reflected in title
- Passage supports inference questions

---

# Core Question Quality

Each unit should include **6–8 core questions**.

Recommended distribution:

- Retrieval ×2
- Vocabulary ×1
- Inference ×2
- Structure ×1
- Author intent ×1

Check:

- Question depends on **reading the passage**
- No generic stems such as:
  - "Which statement is true about the story?"
- No repeated question stems within the unit

---

# MCQ Distractor Rules

Each MCQ must:

- Have **exactly 4 options**
- Options must be **grammatically parallel**
- Distractors must be **plausible**
- Distractors must relate to the passage setting

Bad example:

- on a rocket
- in a supermarket

Good example:

- at the harbour
- beside the lighthouse
- near the pier

---

# Short Answer Questions

Ensure:

- Clear marking point
- Evidence can be taken from passage
- Mark scheme allows paraphrasing

---

# Mark Scheme Quality

Every question must include:

- explicit mark allocation
- acceptable paraphrased answers
- clear distinction between full and partial credit

---

# Additional Question Bank

Additional questions should:

- target specific domains
- not duplicate core questions
- add diagnostic value

Example:

- extra inference questions
- deeper vocabulary analysis

---

# Editorial Status

Units must be labelled:

draft → reviewed → approved
